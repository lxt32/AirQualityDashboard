# 全台空氣品質監測儀錶板

## 📊 功能說明

這是一個專為關心空氣品質的民眾（特別是家中有小孩或過敏人士）設計的互動式儀表板。

### ✨ 核心功能

#### 1. **正常狀態基準線** ✅
- **AQI 基準線**: 50（良好/中等分界）- 顯示為綠色虛線
- **PM2.5 基準線**: 35 µg/m³（良好/中等分界）- 顯示為綠色虛線
- 幫助快速判斷空氣是否超過安全標準

#### 2. **最高數值標記** 🔴
- 在所有圖表和表格中用**紅色突出顯示**
- 紅色背景 + 紅色圓點標記
- 表格中用火焰符號 🔴 標記最高值

#### 3. **警示資訊編碼與文字** 📌
- **六級警示等級系統**:
  * **G** (Good/良好): 綠色 - 適合戶外活動
  * **M** (Moderate/中等): 橙色 - 敏感族群應注意
  * **U-S** (Unhealthy for Sensitive groups/對敏感族群不健康): 深橙 - 敏感族群應避免戶外活動
  * **U** (Unhealthy/不健康): 紅色 - 應避免戶外活動
  * **V-U** (Very Unhealthy/非常不健康): 紫色 - 所有人應避免戶外活動
  * **H** (Hazardous/危害): 深紅 - 危險等級，應留在室內

#### 4. **引導用戶探索儀表板** 🎯
頁面頂部提供多個篩選按鈕：
- **選擇縣市**: 查看特定地區的詳細數據
- **警示等級**: 只顯示特定警示等級的縣市
- **排序方式**: 
  - AQI (從高到低/從低到高)
  - PM2.5 (從高到低/從低到高)
- **重置篩選**: 快速返回全台檢視
- **下載數據**: 將篩選結果匯出為 CSV 檔案

#### 5. **提高信息密度** 📈
儀表板包含多種圖表和統計方式：

**摘要卡片區**
- 每個縣市一張卡片，展示：
  - AQI 值與等級
  - PM2.5 數據
  - 風速、濕度、溫度
  - 健康建議

**數據分析圖表**
- **各縣市 AQI 分布**: 長條圖，帶基準線和數值標籤
- **各縣市 PM2.5 分布**: 長條圖，帶基準線和數值標籤
- **風速與 PM2.5 關係**: 散點圖，展示風速對空氣品質的影響
- **空氣品質等級統計**: 甜甜圈圖，顯示各等級縣市數量

**詳細數據表**
- 所有縣市的完整數據
- 可排序、可篩選
- 最高值行突出顯示

#### 6. **警示程度對應描述** 💡
每個警示等級都有清晰的描述文字：
- **良好**: "空氣品質良好，可正常進行戶外活動"
- **中等**: "空氣品質中等，敏感族群應考慮減少戶外活動"
- **對敏感族群不健康**: "敏感族群應避免戶外活動，一般民眾可正常活動"
- **不健康**: "應避免戶外活動，敏感族群應留在室內"
- **非常不健康**: "所有人都應避免戶外活動"
- **危害**: "危險等級，應留在室內並避免外出"

#### 7. **顏色階層對應** 🎨
嚴格的顏色編碼系統，便於快速視覺識別：

| 警示等級 | 顏色代碼 | 用途 |
|------|-----|--------|
| 良好 | 綠色 (#10b981) | 卡片邊框、圖表顏色、標籤背景 |
| 中等 | 橙色 (#f59e0b) | 卡片邊框、圖表顏色、標籤背景 |
| 對敏感族群不健康 | 深橙 (#f97316) | 卡片邊框、圖表顏色、標籤背景 |
| 不健康 | 紅色 (#ef4444) | 卡片邊框、圖表顏色、標籤背景 |
| 非常不健康 | 紫色 (#7c3aed) | 卡片邊框、圖表顏色、標籤背景 |
| 危害 | 深紅 (#7f1d1d) | 卡片邊框、圖表顏色、標籤背景 |

## 🚀 使用方法

### 1. 打開儀表板
在瀏覽器中打開 `index.html` 檔案。

### 2. 篩選數據
- 使用頂部的下拉菜單選擇特定縣市或警示等級
- 使用排序功能按 AQI 或 PM2.5 值排序
- 點擊「重置篩選」返回初始檢視

### 3. 查看詳細信息
- 頁面自動顯示符合條件的摘要卡片
- 圖表實時更新反映篩選結果
- 詳細數據表展示所有相關信息

### 4. 下載數據
- 點擊「下載數據」按鈕
- 自動下載 CSV 格式的數據檔案
- 可用於進一步分析或記錄

## 📋 數據說明

### AQI (空氣品質指數)
- **範圍**: 0-500
- **正常值**: 0-50（良好）
- **危險值**: 301 以上（危害）

### PM2.5 (細懸浮粒子)
- **單位**: µg/m³（微克每立方公尺）
- **正常值**: 0-35（良好）
- **危險值**: 251 以上（危害）

### 其他參數
- **風速**: m/s（風速越高，PM2.5 濃度通常越低）
- **濕度**: %（相對濕度）
- **溫度**: °C（環境溫度）

本儀表板特別適合以下人群使用：
- 👶 家中有小孩的家庭
- 🤧 有過敏症狀的人士

- 🏥 患有呼吸系統疾病的人士
- 👴 年長人士和其他敏感族群

## 💡 健康建議

### 根據 AQI 值進行活動規劃

| AQI | 等級 | 建議 |
|-----|------|------|
| 0-50 | 良好 | ✅ 可正常進行戶外活動 |
| 51-100 | 中等 | ⚠️ 敏感族群應減少戶外活動 |
| 101-150 | 不健康 (敏感族群) | ⚠️ 敏感族群應避免戶外活動 |
| 151-200 | 不健康 | 🚫 應避免進行戶外活動 |
| 201-300 | 非常不健康 | 🚫 所有人應避免戶外活動 |
| 301+ | 危害 | 🚫 應留在室內 |

## 🔧 技術細節

### 使用技術
- **HTML5**: 頁面結構
- **CSS3**: 現代化設計和響應式佈局
- **JavaScript (ES6+)**: 交互功能
- **Chart.js**: 數據可視化圖表
- **ChartJS Plugin DataLabels**: 圖表標籤顯示

### 文件結構
```
AirQualityDashboard/
├── index.html       # 主頁面
├── data.js          # 數據定義和警示等級定義
├── dashboard.js     # 交互邏輯和圖表初始化
└── README.md        # 本說明檔案
```

## 📝 自定義數據

要更新數據，編輯 `data.js` 檔案中的 `airQualityData` 陣列：

```javascript
const airQualityData = [
    { county: '縣市名稱', aqi: 50, pm25: 20, windSpeed: 2.5, humidity: 65, temperature: 22 },
    // ... 更多數據
];
```
