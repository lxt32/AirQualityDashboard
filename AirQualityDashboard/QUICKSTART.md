# 🚀 快速啟動指南

## 📚 三步開始使用空氣品質監測儀錶板

### 步驟 1️⃣ - 打開檔案
1. 找到 `AirQualityDashboard` 文件夾
2. 用瀏覽器打開 `index.html` 檔案
   - 直接雙擊 `index.html`，或
   - 右鍵選擇「用瀏覽器開啟」

### 步驟 2️⃣ - 檢視全台空氣品質
儀表板會自動顯示：
- ✅ 22 個台灣主要縣市的空氣品質卡片
- 📊 四張關鍵圖表（AQI、PM2.5、風速關係、等級統計）
- 🏆 實時統計數據（平均值、最高值、最低值）
- 📋 詳細數據表格

### 步驟 3️⃣ - 使用篩選功能
探索不同地區的空氣品質：

```
📍 選擇縣市        → 查看該縣市詳細信息
⚠️ 警示等級        → 只顯示特定等級的地區
📊 排序方式        → 按 AQI 或 PM2.5 排序
🔄 重置篩選        → 返回全台檢視
⬇️ 下載數據        → 匯出為 CSV 檔案
```

## 🎯 主要特色速覽

### 🟢 顏色一眼辨識
- 🟢 **綠色** = 良好，可正常活動
- 🟠 **橙色** = 中等，敏感族群注意
- 🟠 **深橙色** = 對敏感族群不健康
- 🔴 **紅色** = 不健康，應避免戶外活動
- 🟣 **紫色** = 非常不健康，嚴禁戶外活動
- 🟥 **深紅** = 危害，應留在室內

1. **各縣市 AQI 分布** - 帶基準線，標明最高值
2. **各縣市 PM2.5 分布** - 帶基準線，標明最高值
3. **風速與 PM2.5 關係** - 散點圖展示相關性
4. **空氣品質等級統計** - 圓形圖顯示各等級占比

| AQI | 等級 | 你應該做... |
|-----|------|-----------|
| 0-50 | 🟢良好 | ✅ 適合戶外運動、遊玩 |

## 🔍 常見使用場景

### 場景 1: 帶小孩去公園前
1. 打開儀表板
2. 查看你家所在地區的 AQI
3. 如果是綠色 ✅，可以放心去公園
4. 如果是紅色 🔴，改為室內活動

### 場景 2: 計畫戶外運動
1. 點擊「排序」 → 選「AQI (從低到高)」
2. 查看哪些地區空氣最好
3. 選擇最優質的地區進行活動

### 場景 3: 監控污染變化
1. 每天同一時間打開儀表板
2. 記錄或下載数据
3. 觀察趨勢，規劃活動

### 場景 4: 過敏人士檢查
1. 篩選「對敏感族群不健康」級別
2. 查看受影響的地區
3. 決定是否需要服用抗過敏藥物或留在室內

## 📥 下載數據使用

點擊「⬇️ 下載數據」按鈕，會下載 CSV 格式的檔案，包含：
- 縣市名稱
- AQI 值及等級
- PM2.5 值及等級
- 風速、濕度、溫度

可用於：
- 📊 在 Excel 中進一步分析
- 📈 建立個人的長期監測記錄
- 📝 與醫生分享過敏和空氣品質的關係

## 🆘 常見問題

**Q: 儀表板為什麼打開是空白的?**
A: 確保在瀏覽器中打開 `index.html`。如果遇到跨域問題，可以：
- 用本地伺服器打開（`python -m http.server 8000`）
- 或在支援的瀏覽器中直接打開

**Q: 可以在手機上使用嗎?**
A: 可以！儀表板已優化為響應式設計，在手機上也能正常顯示。

**Q: 能否保存我的篩選結果?**
A: 目前不能自動保存，但數據下載功能讓你可以保存結果用於分析。

**Q: 數據多久更新一次?**
A: 目前使用示例數據。若連接真實 API，可設置自動更新頻率。
